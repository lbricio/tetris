# tetris
a pygame tetris

## how to play
```bash
python3 tetris.py
```

## commands
| key     | action                        |
| --------- | ---------------------------- |
| `right`   | mofe right        |
| `left`    | move left       |
| `up`      | rotate piece           |
| `down`    | accelerate fall             |

## added feature
- pieces rotate like original tetris
- overtime, player have some frames to adjust/rotate piece before couple
